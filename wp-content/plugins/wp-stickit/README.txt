== Description ==

WP Stickit makes header, sidebar, or anything sticky.


== Installation ==

After downloading and extracting the latest version of WP Stickit:

1. Upload `wp-stickit` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Settings -> WP Stickit in WordPress Admin
4. Place id or class name of sidebar in 'Class Name' field

If you encounter any bugs, or have comments or suggestions, please submit a ticket using support forum.


== Frequently Asked Questions ==

= Can I use multiple class or id ? =

Yes! Just separate them with a comma.

= I placed my widgets class or id. But not working =

Please make sure that you have put the correct symbol for id(#) or class(.) before.
